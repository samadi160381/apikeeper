Quickstart

Three steps to your first authenticated request.

01

Create a key

Sign up and generate a key from the dashboard. It's shown once — copy it immediately.

02

Authenticate

Send it as a bearer token on every request.

03

Read the response

Check X-RateLimit-Remaining and X-Usage-Count headers to track your standing.

curl https://api.apikeeper.dev/v1/ping \
  -H "Authorization: Bearer sk_live_your_key_here"

# 200 OK
{
  "message": "pong",
  "key_id": "29ff8698-4835-...",
  "plan": "free",
  "over_quota_billable": false
}

Error responses

// 401 — missing or invalid key
{ "error": "invalid_api_key" }

// 403 — key lacks required scope
{ "error": "insufficient_scope", "required": "write" }
// 429 — rate limit exceeded
{ "error": "rate_limit_exceeded", "retry_after_ms": 4200 }

// 402 — monthly quota exceeded
{ "error": "quota_exceeded", "used": 1000, "quota": 1000 }
